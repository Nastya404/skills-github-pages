## Жизненный цикл разрешения конфликтов

**Джуниор:**
> — Ой, у меня тут `merge conflict` в ветке... Что делать? Помогите! 

**Миддл:**
> — Спокойно, сейчас аккуратно сделаем `git rebase`, ручками разрешим все конфликты в IDE, прогоним тесты и сделаем чистый `git push`. 

**Сеньор:**
```bash
git add .
git commit -m "у кого упало, тот и чинит"
git push --force
```
*И медленно уходит в закат, ломая сим-карту и меняя паспорт.*
